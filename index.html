Challenge : Keys

Download the zip file and unzip it in terminal by following command :

command : unzip keys.zip

password : hackthebox

Now we have keys.txt file..

In keys.txt have some encoded message

key : hBU9lesroX_veFoHz-xUcaz4_ymH-D8p28IP_4rtjq0=

Token : gAAAAABaDDCRPXCPdGDcBKFqEFz9zvnaiLUbWHqxXqScTTYWfZJcz-WhH7rf_fYHo67zGzJAdkrwATuMptY-nJmU-eYG3HKLO9WDLmO27sex1-R85CZEFCU=

now go to fernet decoder (https://asecuritysite.com/encryption/ferdecode) and decode the above msg

after decoding we have flag

flag : HTB{N0t_A_Fl1g!}
